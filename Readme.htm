Welcome
DM URL Bookmarker is a powerful freeware bookmark URL Organizer program for windows that will to allow you store all your favorite web links.This program offers various special features on how your bookmarks are stored and organized. Most common features include organizing bookmarks in different Categories, Checking bookmark URL status.

This program also offers features that can interact with Internet Explorer that can offer support to add links to the program directly your Web Browser and will also take a screen shot that can make it easier for you to remember the web site.

Other features include exporting bookmarks, Backup and restoring bookmarks, an easy to use config interface that allows you to config the program to your liken. Importing existing databases and much more. This program also offers a powerful set of tools that will allow you to share bookmarks with friends, performing a whois lookup on the bookmark domain name.

At present this program supports all the major web browsers.


Installation
To install this programs simply double click on the setup.exe file. This will then display the main install setup screen from were you will need follow the on screen instructions that will guide the install process.

Requirements

To run this program on your system please refer to the system requirements below:

  • PC with a PII 200 processor or higher.
  • 16 MB of Ram on older systems. 32 MB recommenced if using Widnows XP.
  • VGA 16 Bit display adapter or better.
  • Internet Explorer 5.0 or better.
  • Windows98, Windows 2000, Windows XP or better.
  • 3 MB of free hard disk space.
